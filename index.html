📱 iOS Shortcut Setup
5-minute setup. Every evening Kharcha auto-collects your bank SMSes and lets you review them in 30 seconds.
1
Open Shortcuts app on iPhone → tap "+" to create a new Shortcut → name it "Kharcha SMS"
2
Add Action: search "Get Messages" → add it. Set Filter by Sender → Contains → type your bank's SMS sender ID.

Common sender IDs: HDFCBK SBIINB ICICIB AXISBK KOTAKB PAYTM ATMMSG

Set Sort: Most Recent. Set Limit: 30
3
Repeat step 2 for each bank you use. Tap "+" to add another "Get Messages" action per bank.
4
Add Action: search "Combine Text". In the input tap the variable and select all message results. Set separator: New Lines
5
Add Action: "Copy to Clipboard". Set input to the Combined Text from step 4.
6
Add Action: "Show Notification". Title: Kharcha Ready. Body: Open app → Review tab → Paste to log expenses
7
Automate it: Go to Automation tab → "+" → Time of Day → set 8:00 PM, Daily → select your Kharcha SMS shortcut → Turn off "Ask Before Running"
Evening flow: 8 PM notification arrives → tap to open Kharcha → go to Review tab → tap the text box → long-press → Paste → tap Scan. Done in 30 seconds. 🚀

AI learning: Every time you confirm a transaction, Kharcha learns that merchant's category. Next time it auto-selects the right one!
